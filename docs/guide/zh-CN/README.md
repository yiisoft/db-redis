Yii 2 Redis 缓存，会话和活动记录
===============================================

此扩展为 Yii2 framework 提供了 [redis](https://redis.io/) 键-值存储的支持。
包含了一个 `Cache` 和 `Session` 存储句柄并且实现了 `ActiveRecord` 模式，允许
你存储活动记录到 redis 中。
